## Implementation Notes

### Bar Width Calculations
```
Base resolution:
- Minimum bar width: 2px
- Minimum gap: 1px
- Formula: bar_width = (chart_width - (gaps * gap_width)) / num_bars

Scaled resolutions:
- 2x: 4px bars, 2px gaps
- 3x: 6px bars, 3px gaps
- 4x: 8px bars, 4px gaps
```

### Color Threshold Logic
```rust
fn get_bar_color(value: f32, max_value: f32) -> u8 {
    let percentage = (value / max_value) * 100.0;
    match percentage {
        p if p < 60.0 => 6,  // Green
        p if p < 80.0 => 3,  // Orange
        _ => 2,              // Red
    }
}
```

### Grid Alignment
All bars must align to the 8×8 pixel grid:
- X position: Multiple of 8
- Width: Multiple of 2 (minimum)
- Height: Exact pixel values, no fractions
