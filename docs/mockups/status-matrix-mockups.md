## Implementation Details

### Dot Rendering Function
```rust
fn render_status_dot(x: u16, y: u16, status: Status, scale: u8) {
    let color = match status {
        Status::Ok => 5,        // Lime
        Status::Warning => 4,   // Yellow
        Status::High => 3,      // Orange
        Status::Critical => 2,  // Red
        Status::Error => 1,     // Purple
        Status::Disabled => 14, // Gray
    };
    
    let size = 3 * scale; // Base 3x3, scales up
    for dy in 0..size {
        for dx in 0..size {
            plot_pixel(x + dx, y + dy, color);
        }
    }
}
```

### Grid Layout Calculations
```
Cell width = max(host_name_width, 40) + (num_services * 16)
Cell height = 8 pixels (base)
Dot spacing = 1 pixel minimum
Border = 1 pixel

Total width = cell_width + (num_services * (dot_size + spacing))
Total height = header_height + (num_hosts * cell_height)
```

### Performance Optimizations
- Pre-render all dot states (6 colors × 4 scales = 24 sprites)
- Use dirty rectangles for blinking states only
- Cache rendered text strings
- Batch draw operations by color
