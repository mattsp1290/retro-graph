## Color and Pattern Reference

### Fill Patterns Used
```
████ = 100% solid fill
░░░░ = 75% dither pattern
▒▒▒▒ = 50% dither pattern
····· = 25% dither pattern
```

### Line Styles at Different Scales
```
1x: ─ (single pixel)
2x: ══ (double thickness)
3x: ≡≡≡ (triple thickness)
4x: ████ (quad thickness)
```
